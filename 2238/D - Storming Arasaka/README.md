Storming Arasaka</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><div class="epigraph"><div class="epigraph-text">Hah, you just discovered what it takes to become a legend.</div><div class="epigraph-source">— Cyberpunk 2077</div></div><p>You and Johnny Silverhand decided to storm Arasaka together. After making your way through the guards, you reached Mikoshi — but in order to connect to it, you need to hack the main server.</p><p>The password to the server is formed as follows. There is a secret number $$$n$$$. Consider all of its positive divisors except $$$1$$$, <span class="tex-font-style-bf">but the divisor equal to </span> $$$\mathbf{n}$$$ <span class="tex-font-style-bf"> is considered</span>, and partition <span class="tex-font-style-bf">all</span> of them into several nonempty layers $$$L_1, L_2, \ldots, L_k$$$. A partition is called good if two conditions are satisfied:</p><ul> <li> for any divisor $$$x$$$ from layer $$$L_i$$$, all of its divisors, except $$$1$$$ and $$$x$$$, lie only in the layers $$$L_1, L_2, \ldots, L_{i-1}$$$; </li><li> in each layer, all numbers can be ordered into a chain so that any two neighboring numbers in this chain have GCD$$$^{\text{∗}}$$$ greater than $$$1$$$. </li></ul><p>The length of the password is defined as the number of layers $$$k$$$. For the security of the layers, their number must be as small as possible.</p><p>Fortunately, Arasaka has not changed $$$n$$$ since Johnny's time, and he remembers several possible values of this number. For each of them, help V and Johnny determine the minimum possible number of layers.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$$$$\gcd(x, y)$$$ denotes the <a href="https://en.wikipedia.org/wiki/Greatest_common_divisor">greatest common divisor (GCD)</a> of integers $$$x$$$ and $$$y$$$.</p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). you.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single number — the minimum number of layers.</p></div><div
